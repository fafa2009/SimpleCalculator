# SimpleCalculator

A simple web calculator that performs arithmetic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear screen

## How to Run
Open index.html in any web browser.

SDLC
1. Planning Phase
Project Title: SimpleCalculator
 Objective: To design and implement a web-based calculator for basic arithmetic operations.
 Target Users: Students and general users.
 Platform: Web Application.
 Tools Used: VS Code, Web Browser, GitHub.
Feasibility Study:
 - Technical: Uses standard web technologies
 - Economic: Free tools
 - Operational: Very easy to use

2. Requirements Analysis
Functional Requirements
- User can enter numbers
- User can choose an arithmetic operator
- User can get the calculated result
- User can clear the screen
Non-Functional Requirements
- Simple interface
- Fast response
- Works on any browser
